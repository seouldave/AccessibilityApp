Instructions

The aim of this application is to measure accessibility to health facilities based on a user-defined travel-time threshold. The target users are healthcare workers, or resource allocation decision-makers, with limited GIS knowledge. All analysis is carried out on the server. The main requirements to perform this analysis is a computer with internet access, and knowledge of the locations of health facilities. The length of processing time may vary depending on the number of facilities analysed.
To assist in finding locations on the map, it may be useful to use the slider at the top of the righthand panel to change the transparency of the satellite image/map used as a background image.
Please see the information below for instructions.

  1. Choose a country

    Note: This application is currently in its Alpha stage, and covers only 7 countries in Africa

  2. Choose a population group

    This application is currently focused on access to EmONC facilities. Target populations used in the application are number of:

    • Pregnant women
    • Number of births
    • Number of women of childbearing age (15-49)
  3. Choose a mode of transport
  4. Upload health facilities

    Health facilities can be uploaded using a .csv file of coordinates or by double-clicking their positions on the map. Both methods can be used together

    Upload csv file

    • Create and save csv file with WG84 coordinates. Longitude values in the first column, latitude values in the second column
    • Click the 'Browse' button to choose the csv file
    • Click the 'Read file' button and the 'Add uploaded points' to add the points to the map

    Manually add points

    • Click the 'Manually add points' button
    • Point to the location of health facilities on the map and double-click to add the point

    To remove points, single-click on the point to remove and click the 'Remove' button that appears next to the clicked point.

  5. Input travel-time threshold in hours
  6. Submit data for processing

    Once all of the health facility points have been added, click the 'Submit' button to send the data to be processed. The data should be processed at approximately 1 second per health facility. A raster will be returned colour-coded according to the time predicted to travel from the respective pixels to the closest (shortest travel time) health facility. Further analyses can be carried out be clicking the 'Restart' button. Should be page not fully refresh, try to reload the page by pressing the 'Ctrl + F5' buttons on your keyboard.

Note: A panel will appear on the top left of the screen as layers are added to the map. The panel will allow the user to choose the layer to be shown on the top of the stack. The top layer's legend will be displayed in the bottom of the legend. A layer cannot be selected in this panel until it has been added to the map. The final output layer will not be selectable until the layer has been fully processed and returned from the server.
To view the map in fullscreen mode, click the icon in the top-right corner of the screen.
Once the output has been returned to the website, a clickable element will appear to the left of the main interface panel. Click this to toggle the population tables.
